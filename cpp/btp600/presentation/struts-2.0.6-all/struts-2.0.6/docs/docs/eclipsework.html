About Eclipse
Eclipse is a platform for the development of tools and rich client applications.
  • EclipseWork is a third-party project created by Ricardo Lecheta. It is an Eclipse plugin for WebWork. Future plans include abstracting the plugin and providing for a plugin base that can be used to build both the Eclipse plugin and a new WW:IDEA Plugin.
