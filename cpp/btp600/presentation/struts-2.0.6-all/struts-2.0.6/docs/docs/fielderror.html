fielderror

Description

Render field errors if they exists. Specific layout depends on the particular theme.

Name

Required

Default

Evaluated

Examples

<!-- example 1 -->
   <s:fielderror />

   <!-- example 2 -->
   <s:fielderror>
        <s:param>field1</s:param>
        <s:param>field2</s:param>
   </s:fielderror>
   <s:form .... >>
      ....
   </s:form>

   OR

   <s:fielderror>
         <s:param value="%{'field1'}" />
         <s:param value="%{'field2'}" />
   </s:fielderror>
   <s:form .... >>
      ....
   </s:form>
