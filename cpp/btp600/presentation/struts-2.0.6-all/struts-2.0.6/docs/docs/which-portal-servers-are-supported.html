List of supported portal servers

The following portal servers are known to work with the framework's portlet integration:

  • Apache Jetspeed 2
  • eXo Portal 1.1
  • Gridsphere 2.1
  • IBM WebSphere Portal 5.1
  • JBoss-Portal 2.0
  • JBoss-Portal 2.2
  • Liferay-3.6.1
  • Pluto 1.0.1

If your portal server is not listed here, and you would like to contribute required configuration files and additional deployment descriptors, please file an issue and annex the material.
